# 🚀 Smart Task Manager - Django

Ce projet est une application web de gestion de tâches développée dans le cadre du volet Créativité . Elle permet d'organiser ses activités quotidiennes de manière efficace.

## ✨ Fonctionnalités clés
* **Gestion dynamique :** Ajout, modification et suppression de tâches.
* **Organisation par catégories :** Filtres pour séparer le travail, les loisirs et les études.
* **Interface Responsive :** Design moderne utilisant **Bootstrap** pour une utilisation fluide sur PC et mobile.
* **Déploiement Cloud :** Application entièrement en ligne via PythonAnywhere.

## 🛠️ Technologies utilisées
* **Backend :** Python 3.10 & Django
* **Frontend :** HTML5, CSS3, Bootstrap 5
* **Base de données :** SQLite
